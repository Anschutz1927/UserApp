# UserApp
*Training


To install this app on your phone you need:
1. Straight arms
2. Android studio 2

At Android Studio:
1. Create new empty project. 
2. Log in to your account to GitHub in settings. 
3. Then VCS/Checkout from VersionControl/GitHub and clone the repository on your PC.
4. Build/Make Project
5. Run/Run 'App'
*notice: 
*enable unknown sources in settings in your phone; enable Android debugging inDeveloper options in Settings.
